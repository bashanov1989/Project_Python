# Исследовательский анализ данных (EDA) на Python

В проекте использованы современные библиотеки (Pandas, NumPy, Matplotlib, Seaborn) для анализа больших датасетов.

**Основные этапы:**
- **Data Cleaning:** Очистка данных от пропусков, дубликатов и аномалий.
- **EDA:** Выявление скрытых закономерностей и корреляций между показателями.
- **Визуализация:** Построение графиков для наглядного представления статистических выводов.
